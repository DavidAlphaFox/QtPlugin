# Panda Qt5 Plugin

Unify Qt application style of pandaOS.

## Dependencies

`sudo pacman -S gcc extra-cmake-modules qt5-base qt5-tools qt5-x11extras libdbusmenu-qt5 libxcb`

## Build

```shell
mkdir build
cd build
cmake ..
make
sudo make install
```
